이번 세션에서 한 작업을 돌아보고 마무리하라.

1. 오늘 생성/수정한 wiki 페이지를 나열.
2. `wiki/log.md`에 오늘 항목이 빠짐없이 기록됐는지 확인하고 보완 (CLAUDE.md 로그 포맷 준수).
3. `wiki/hot.md`를 ~500단어로 갱신: 현재 포커스 · 열린 질문 · 오늘의 결정 · 다음에 할 일.
4. `VAULT-INDEX.md`의 Active 체크박스 상태 업데이트.
5. 변경이 있으면 `git add -A && git commit -m "close: {오늘 요약}"` 제안.

`sources/`(raw)는 절대 건드리지 않는다.
